## Getting Started 🚀 

### Clonando repositorio

```js
git clone https://gitlab.com/matrix-campus/prueba-tecnica-rafael-punk-api.git
```

### Comenzando proyecto Angular

cd prueba-tecnica-rafael-punk-api

npm install

ng serve
